# tarefa2_Xabier_FerreiroSanmartin

## Este é o repositorio creado polo alumno Xabier Ferreiro Sanmartín para a tarefa 2 do módulo de contornas de desarollo do curso de desenvolvemento en python.

O proceso de realización de esta tarefa foi:
1. Crear un repositorio e clonalo no meu entorno local
2. Instalar venv con pip e activar o entorno virtual na carpeta do proxecto
3. Asegurarme que no .gitignore que xerei ao crear o repositorio están incluidas as liñas necesarias para que non se suban os arquivos do entorno virtual ao repositorio. Neste caso github permitiume xerar un gitignore específico para proxectos en python que entre moitas outras cousas exclue o entorno virtual (a liña "venv/" no apartado "Envirioments"), polo que deixei o .gitignore intacto.
4. Instalar o paquete faker
5. Crear o arquivo requirements.txt co comando pip freeze > requirements.txt
6. Programar o script en vscode asegurandome de que sempre que o executo estou dentro do contorno virtual
7. Unha vez programado e testeado, pushear os cambios ao repositorio e cerrar o entorno virtual.
